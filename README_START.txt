Как обновить проект:
1. Удалите старую папку проекта или замените файлы полностью.
2. В терминале в папке проекта выполните:
   docker compose down
   docker compose build --no-cache
   docker compose up
3. Если браузер всё равно показывает старый вид — нажмите Ctrl+F5.

Открывать:
- http://localhost:5500
- http://localhost:8000/docs
